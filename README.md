# Egypt Tourism App 🇪🇬

A modern, immersive React Native application for exploring Egypt's top tourist destinations. featuring a premium design, multi-language support (English/Arabic), and smooth animations.

## ✨ Features

- **Discover Places**: Browse top attractions with filtering by category (Historical, Beach, Nature, etc.).
- **Multi-language Support**: Full support for English and Arabic (RTL layouts).
- **Interactive Map**: View locations on a map (integration ready).
- **Trip Planner**: Organize your itinerary.
- **Favorites**: Save places to your wishlist.
- **Dark Mode**: (Coming Soon)
- **Smooth Animations**: Powered by `react-native-reanimated`.
- **Premium UI**: Custom fonts, glassmorphism effects, and responsive layout.

## 🛠 Tech Stack

- **Framework**: [Expo](https://expo.dev/) (React Native)
- **Navigation**: React Navigation (Stack & Tabs)
- **Styling**: StyleSheet, Lucide Icons, Expo Vector Icons
- **State Management**: React Context API
- **Persistence**: AsyncStorage
- **Animations**: React Native Reanimated

## 🚀 Getting Started

### Prerequisites

- Node.js installed
- Expo Go app on your phone (or Android Studio/Xcode for emulator)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/MidoShehaa/EgyptTourismMobile.git
   cd EgyptTourismMobile
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   npx expo install
   ```

3. **Start the app**
   ```bash
   npx expo start --clear
   ```

4. **Run on device**
   - Scan the QR code with Expo Go (Android) or Camera (iOS).
